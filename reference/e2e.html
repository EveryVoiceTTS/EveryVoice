E2E#

Command Line Interface#

everyvoice#

everyvoice [OPTIONS] COMMAND [ARGS]...

e2e#

End-to-end training: jointly train the FastSpeech2 and HiFiGAN networks

everyvoice e2e [OPTIONS] COMMAND [ARGS]...
train#
everyvoice e2e train [OPTIONS]

Options

-n, --name <name>#
Options:

training | lj_training | lj | base

-c, --config <config_args>#
-p, --config-path <config_path>#
-a, --accelerator <accelerator>#

Uses PyTorch Lightning Accelerators: https://pytorch-lightning.readthedocs.io/en/stable/extensions/accelerator.html

Default:

auto

-d, --devices <devices>#

The number of GPUs on each node

Default:

auto

-n, --nodes <nodes>#

The number of nodes on your machine

Default:

1

-s, --strategy <strategy>#

The strategy for data parallelization: https://pytorch-lightning.readthedocs.io/en/stable/accelerators/gpu_intermediate.html

Default:

ddp
