Vocoder#

HiFiGAN/iSTFT-Net#

Source Code: https://github.com/roedoejet/HiFiGAN_iSTFT_lightning

This vocoder is based on the HiFiGAN/iSTFT-Net neural vocoders.

Command Line Interface#

smts#

smts [OPTIONS] COMMAND [ARGS]...
hifigan#
smts hifigan [OPTIONS] COMMAND [ARGS]...
preprocess#
smts hifigan preprocess [OPTIONS]

Options

--name <name>#
Options

base | openslr | lj

-d, --data <data>#
Options

audio | spec

-c, --config-args <config_args>#
-p, --config-path <config_path>#
-o, --output <output_path>#
Default

processed_filelist.psv

-C, --cpus <cpus>#
Default

2

-O, --overwrite#
Default

False

-D, --debug#
Default

False

synthesize#
smts hifigan synthesize [OPTIONS]

Options

-i, --input <data_path>#
-m, --model <generator_path>#
train#
smts hifigan train [OPTIONS]

Options

-n, --name <name>#
Options

base | openslr | lj

-c, --config <config_args>#
-p, --config-path <config_path>#
-a, --accelerator <accelerator>#

auto

auto
