Vocoder#

HiFiGAN/iSTFT-Net#

Source Code: roedoejet/HiFiGAN_iSTFT_lightning

This vocoder is based on the HiFiGAN/iSTFT-Net neural vocoders.

Command Line Interface#

everyvoice#

everyvoice [OPTIONS] COMMAND [ARGS]...
hifigan#
everyvoice hifigan [OPTIONS] COMMAND [ARGS]...
preprocess#
everyvoice hifigan preprocess [OPTIONS]

Options

--name <name>#
Options:

base | openslr | lj

-d, --data <data>#
Options:

audio | spec

-c, --config-args <config_args>#
-p, --config-path <config_path>#
-o, --output <output_path>#
Default:

processed_filelist.psv

-C, --cpus <cpus>#
Default:

2

-O, --overwrite#
Default:

False

-D, --debug#
Default:

False

synthesize#
everyvoice hifigan synthesize [OPTIONS]

Options

-i, --input <data_path>#
-m, --model <generator_path>#
train#
everyvoice hifigan train [OPTIONS]

Options

-n, --name <name>#
Options:

base | openslr | lj

-c, --config <config_args>#
-p, --config-path <config_path>#
-a, --accelerator <accelerator>#

Uses PyTorch Lightning Accelerators: https://pytorch-lightning.readthedocs.io/en/stable/extensions/accelerator.html

Default:

auto

-d, --devices <devices>#

The number of GPUs on each node

Default:

auto

-n, --nodes <nodes>#

The number of nodes on your machine

Default:

1

-s, --strategy <strategy>#

The strategy for data parallelization: https://pytorch-lightning.readthedocs.io/en/stable/accelerators/gpu_intermediate.html

Default:

ddp
